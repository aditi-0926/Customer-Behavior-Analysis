# Customer Trends Data Analysis Using SQL, Python & Power BI
Complete Data Analytics Portfolio Project with end-to-end industry standard Data Analysis of Customer Shopping Trends from Retail Data using SQL, Python and Power BI.

## Project Overview

This project focuses on analyzing customer purchasing behavior and business performance using SQL, Python, and Power BI. The objective is to extract meaningful insights from customer transaction data, identify purchasing trends, and support data-driven business decisions.

The project demonstrates a complete analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL-based querying, and dashboard creation.

---

## Objectives

* Analyze customer demographics and purchasing patterns.
* Identify top-performing products and categories.
* Measure sales and revenue trends.
* Understand customer behavior and engagement.
* Create interactive dashboards for business decision-making.

---

* **SQL (PostgreSQL)** – Data storage and querying

* **Power BI** – Interactive dashboard development

## Project Workflow

### 1. Data Collection

* Imported customer transaction dataset.
* Examined data structure and attributes.

### 2. Data Cleaning

* Removed duplicates.
* Handled missing values.
* Corrected data types.
* Validated data consistency.

### 3. Exploratory Data Analysis (EDA)

* Customer distribution analysis.
* Product performance analysis.
* Sales trend analysis.
* Revenue analysis.
* Correlation analysis.

### 4. SQL Analysis

Performed various SQL queries to:

* Calculate total sales and revenue.
* Identify top-selling products.
* Analyze customer purchasing behavior.
* Determine monthly sales trends.
* Generate business performance reports.

### 5. Power BI Dashboard

Developed an interactive dashboard featuring:

* Total Sales Overview
* Revenue Analysis
* Customer Insights
* Product Performance
* Sales Trends
* Key Performance Indicators (KPIs)
